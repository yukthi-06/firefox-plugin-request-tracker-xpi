# Walkthrough - Use app-icon.png as Plugin Icon

## Changes Made

### Extension Metadata
- Updated [manifest.json](file:///x:/Projects_X/0_Active/A_XPI_Firefox_Plugin/firefox-plugin-request-tracker-xpi_GITHUB/extension/manifest.json):
  - Changed default action icon from `"icons/icon.svg"` to `"icons/app-icon.png"`.
  - Added a root `"icons"` object mapping sizes `48`, `96`, and `128` to `"icons/app-icon.png"`.

### Build Script
- Updated [build.bat](file:///x:/Projects_X/0_Active/A_XPI_Firefox_Plugin/firefox-plugin-request-tracker-xpi_GITHUB/build.bat):
  - Added a build step to copy `app-icon.png` from the root of the project to `extension/icons/app-icon.png` automatically whenever the extension is packaged.

## Verification
- To package the extension and copy the icon:
  1. Run `build.bat` on Windows.
  2. The script will automatically copy `app-icon.png` into `extension/icons/app-icon.png` and package it into the XPI file under the `output_xpi/` folder.
