# User Prompt
on the left side of the "Start Capture" or "Stop Capture" button, have a circle which can show GREEN color for Capturing active and RED color for Capturing Stopped status

## Goal

Add a visual capturing status indicator circle directly to the left of the "Start Capture" / "Stop Capture" button, showing:
- **GREEN** (#00ff00) when capturing is active.
- **RED** (#ff0000) when capturing is stopped.

### Script Logic

#### [MODIFY] [logger.js](file:///x:/Projects_X\0_Active\A_XPI_Firefox_Plugin\firefox-plugin-request-tracker-xpi_GITHUB\extension\logger.js)
- Update `updateTrackingStateUI()` to toggle `active`/`stopped` classes on the `#capture-status-indicator` element based on `trackingActive`.

```javascript
function updateTrackingStateUI() {
  const btn = document.getElementById('toggle-capture-btn');
  const indicator = document.querySelector('.pulse-indicator');
  const statusCircle = document.getElementById('capture-status-indicator');
  
  if (trackingActive) {
    btn.textContent = 'Stop Capture';
    btn.className = 'btn btn-secondary';
    indicator.classList.remove('inactive');
    if (statusCircle) {
      statusCircle.classList.remove('stopped');
      statusCircle.classList.add('active');
    }
  } else {
    btn.textContent = 'Start Capture';
    btn.className = 'btn btn-primary';
    indicator.classList.add('inactive');
    if (statusCircle) {
      statusCircle.classList.remove('active');
      statusCircle.classList.add('stopped');
    }
  }
}
```
